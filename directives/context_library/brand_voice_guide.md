# Brand Voice Guide (Placeholder)

## Tone and Style
- **Professional yet Accessible**: Use clear, concise language. Avoid overly academic jargon.
- **Empowering**: Focus on how the reader can achieve results.
- **Authoritative**: Back up claims with data and clear logic.

## Formatting Preferences
- Use bullet points for readability.
- Keep paragraphs under 4 lines.
- Use bold text for key takeaways.
